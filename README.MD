### py-personal-wallet
Program for recording personal income/expenses
### Features
 - Show Balance
 - Add Record
 - Edit Record
 - Search Record
 - Exit

### How to run ?
1. Install Python to your system
2. Activate the virtual environment
    ```powershell
    # Windows
    python -m venv venv
    .\venv\Scripts\activate
    ```
    ```sh
    # Linux
    python3 -m venv venv
    source venv/bin/activate
3. Run code with command:
   ```powershell
    #Windows
    py main.py
    ```
    
    ```sh
    #Linux
    python3 main.py
    ```
